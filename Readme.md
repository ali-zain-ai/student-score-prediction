# Student Score Prediction

A machine learning project that predicts student scores based on academic and behavioral factors.

## Features
- Trained ML regression model
- Flask web app for predictions
- Scaled inputs using saved transformer

## Project Structure
- app/ → Flask application
- models/ → Trained model and scaler
- notebooks/ → Model training notebook
- data/ → Dataset

## Tech Stack
- Python
- Scikit-learn
- Flask
- Pandas

## How to Run
pip install -r requirements.txt  
python app/app.py
